---
title: "Delete Email Account"
api: "DELETE https://server.smartlead.ai/api/v1/email-accounts/{email_account_id}"
description: "Permanently delete an email account and remove from all campaigns"
---

## Path Parameters

<ParamField path="email_account_id" type="number" required>
  The email account id
</ParamField>



## Query Parameters

<ParamField query="api_key" type="string" required>
  Your SmartLead API key
</ParamField>



## Request Body

<Note>No request body required</Note>

<RequestExample>

```bash cURL
curl -X DELETE "https://server.smartlead.ai/api/v1/email-accounts/{email_account_id}?api_key=YOUR_KEY"
```

```python Python
import requests

API_KEY = "YOUR_API_KEY"

response = requests.delete(
    "https://server.smartlead.ai/api/v1/email-accounts/{email_account_id}",
    params={"api_key": API_KEY}
)

result = response.json()
print(result)
```

```javascript JavaScript
const API_KEY = 'YOUR_API_KEY';

const response = await fetch(
  `https://server.smartlead.ai/api/v1/email-accounts/${email_account_id}?api_key=${API_KEY}`
);

const result = await response.json();
console.log(result);
```

</RequestExample>
