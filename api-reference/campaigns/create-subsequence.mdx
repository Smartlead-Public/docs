---
title: "Create Subsequence Campaign"
api: "POST https://server.smartlead.ai/api/v1/campaigns/{campaign_id}/create-subsequence"
description: "Create a child campaign (subsequence) with conditional logic"
---

<Note>
Create branching campaign logic by defining subsequences that triggers based on lead behavior. Essential for sophisticated nurture workflows.
</Note>

## Path Parameters

<ParamField path="campaign_id" type="number" required>
  Parent campaign ID
</ParamField>

## Query Parameters

<ParamField query="api_key" type="string" required>
  Your SmartLead API key
</ParamField>
