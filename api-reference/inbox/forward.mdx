<Note>
Forwards a complete email thread to colleagues or external contacts. Maintains original message context. Requires message_id, stats_id, and recipient emails.
</Note>

## Overview

Forwards an email conversation from a campaign to one or more email addresses. Useful for:
- Delegating leads to team members
- Sharing conversations with sales team
- Routing to appropriate departments
- Escalating important conversations
- Collaborating on responses

The forwarded email includes the original message thread and context from the campaign.

## Path Parameters

<ParamField path="campaign_id" type="number" required>
  The ID of the campaign containing the email to forward
</ParamField>

## Query Parameters

<ParamField query="api_key" type="string" required>
  Your SmartLead API key for authentication
</ParamField>

## Request Body

<ParamField body="message_id" type="string" required>
  The message ID of the email to forward
</ParamField>

<ParamField body="stats_id" type="string" required>
  The email statistics ID associated with this message
</ParamField>

<ParamField body="to_emails" type="string" required>
  Comma-separated list of email addresses to forward to
  
  Example: "colleague@company.com,manager@company.com"
</ParamField>

<RequestExample>

```bash cURL
curl -X POST "https://server.smartlead.ai/api/v1/campaigns/123/forward-email?api_key=YOUR_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "message_id": "msg_abc123",
    "stats_id": "stat_xyz789",
    "to_emails": "colleague@company.com,manager@company.com"
  }'
```

```python Python
import requests

API_KEY = "YOUR_API_KEY"
campaign_id = 123

payload = {
    "message_id": "msg_abc123",
    "stats_id": "stat_xyz789",
    "to_emails": "colleague@company.com,manager@company.com"
}

response = requests.post(
    f"https://server.smartlead.ai/api/v1/campaigns/{campaign_id}/forward-email",
    params={"api_key": API_KEY},
    json=payload
)

result = response.json()
print("Email forwarded successfully" if response.status_code == 200 else "Error")
```

```javascript JavaScript
const API_KEY = 'YOUR_API_KEY';
```

## Response Codes

<ResponseField name="200" type="Success">
  Email forwarded successfully
</ResponseField>

<ResponseField name="400" type="Bad Request">
  Invalid email addresses or missing required fields
</ResponseField>

<ResponseField name="401" type="Unauthorized">
  Invalid or missing API key
</ResponseField>

<ResponseField name="404" type="Not Found">
  Campaign or message not found
</ResponseField>

<ResponseField name="422" type="Validation Error">
  Invalid message_id, stats_id, or email format

<ResponseField name="500" type="Internal Server Error">
  Server error occurred while forwarding
</ResponseField>

## Use Cases

### Forward to Team Member
```python
# Forward hot lead to sales team
response = requests.post(
    f"{base_url}/campaigns/{campaign_id}/forward-email",
    params={"api_key": API_KEY},
    json={
        "message_id": interested_lead_msg_id,
        "stats_id": interested_lead_stats_id,
        "to_emails": "sales@company.com"
    }
)
```

### Forward to Multiple Recipients
```python
# Forward to multiple team members
to_emails = "sales@company.com,manager@company.com,support@company.com"

response = requests.post(
    url,
    params={"api_key": API_KEY},
    json={
        "message_id": msg_id,
        "stats_id": stats_id,
        "to_emails": to_emails  # Comma-separated
    }
)
```

## Getting Message and Stats IDs

To forward an email, you need the `message_id` and `stats_id`. Get these from:

1. **Master Inbox endpoints** - All inbox views return these IDs
2. **Campaign statistics** - Email stats include these IDs
3. **Webhook events** - Events include message and stats IDs

Example from inbox:
```json
{
  "email_stats_id": "stat_xyz789",  // This is your stats_id
  "message_id": "msg_abc123",       // This is your message_id
  "lead_email": "lead@example.com"
}
```
