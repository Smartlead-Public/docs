<Note>
Forward email threads to colleagues or external recipients. Maintains original context and thread history.
</Note>

## Overview

Forwards an email or entire thread to specified recipients. Useful for collaboration, escalation, and sharing important conversations.

**Use Cases:**
- Escalate to manager/supervisor
- Share with team members
- Forward to subject matter expert
- Loop in decision makers
- External referrals

## Query Parameters

<ParamField query="api_key" type="string" required>
  Your SmartLead API key
</ParamField>

## Request Body

<ParamField body="campaignId" type="number" required>
  Campaign ID
</ParamField>

<ParamField body="emailStatsId" type="string" required>
  Email stats ID of the message to forward
</ParamField>

<ParamField body="emailThreadId" type="number">
  Thread ID (optional, for forwarding entire thread)
</ParamField>

<ParamField body="forwardEmailSubject" type="string" required>
  Subject line for the forwarded email
</ParamField>

<ParamField body="forwardEmailBody" type="string" required>
  Body content for the forward message
</ParamField>

<ParamField body="forwardToEmailIds" type="string" required>
  Comma-separated list of recipient email addresses
</ParamField>

```python Python
import requests

API_KEY = "YOUR_API_KEY"

def forward_email(campaign_id, email_stats_id, to_emails, subject, body, thread_id=None):
    """Forward an email to other recipients"""
    payload = {
        "campaignId": campaign_id,
        "emailStatsId": email_stats_id,
        "forwardEmailSubject": subject,
        "forwardEmailBody": body,
        "forwardToEmailIds": to_emails
    }
    
    if thread_id:
        payload["emailThreadId"] = thread_id
    
    response = requests.post(
        "https://server.smartlead.ai/api/v1/email-campaigns/forward-reply-email",
        params={"api_key": API_KEY},
        json=payload
    )
    
    return response.json()

# Forward hot lead to manager
forward_email(
    campaign_id=12345,
    email_stats_id="abc-123",
    to_emails="manager@company.com",
    subject="FW: High-Value Lead",
    body="This lead expressed strong interest and has budget. Please review."
)

# Forward to multiple recipients
forward_email(
    campaign_id=12345,
    email_stats_id="abc-124",
    to_emails="sales@company.com,support@company.com",
    subject="FW: Technical Question",
    body="Can someone from the technical team help with this?"
)
```

```javascript JavaScript
const API_KEY = 'YOUR_API_KEY';

async function forwardEmail(campaignId, emailStatsId, toEmails, subject, body, threadId = null) {
  const payload = {
    campaignId,
    emailStatsId,
    forwardEmailSubject: subject,
    forwardEmailBody: body,
    forwardToEmailIds: toEmails,
    ...(threadId && { emailThreadId: threadId })
  };
  
  const response = await fetch(
    `https://server.smartlead.ai/api/v1/email-campaigns/forward-reply-email?api_key=${API_KEY}`,
    {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify(payload)
    }
  );
  
  return response.json();
}

// Forward to manager
await forwardEmail(
  12345,
  'abc-123',
  'manager@company.com',
  'FW: Enterprise Lead',
  'Please review this enterprise opportunity'
);
```

## Common Workflows

### Escalation to Manager
```python
def escalate_to_manager(email_stats_id, lead_email, reason):
    forward_email(
        campaign_id=get_campaign_id(email_stats_id),
        email_stats_id=email_stats_id,
        to_emails="manager@company.com",
        subject=f"FW: Escalation - {lead_email}",
        body=f"Escalation Reason: {reason}\n\nPlease review this conversation."
    )
```

### Team Collaboration
```python
def share_with_team(email_stats_id, team_emails, note):
    forward_email(
        campaign_id=get_campaign_id(email_stats_id),
        email_stats_id=email_stats_id,
        to_emails=",".join(team_emails),
        subject="FW: Team Review Needed",
        body=f"Team Note: {note}\n\nPlease provide your input."
    )
```
