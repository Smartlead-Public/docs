<Note>
Send replies to leads directly from the inbox. Maintains thread continuity and tracks all responses.
</Note>

## Overview

Sends a reply email to a lead, maintaining the email thread. Can send immediately or schedule for later.

**Key Features:**
- Thread continuity maintained
- Optional scheduling
- CC/BCC support
- Attachments support
- Signature inclusion
- Reply tracking in conversation history

```python Python
import requests

API_KEY = "YOUR_API_KEY"

def send_reply(campaign_id, email_stats_id, body, schedule_time=None, add_signature=True):
    payload = {
        "campaignId": campaign_id,
        "emailStatsId": email_stats_id,
        "emailBody": body,
        "addSignature": add_signature
    }
    
    if schedule_time:
        payload["scheduledTime"] = schedule_time
    
    response = requests.post(
        "https://server.smartlead.ai/api/v1/email-campaigns/send-email-thread",
        params={"api_key": API_KEY},
        json=payload
    )
    
    return response.json()

# Send immediate reply
send_reply(12345, "abc-123", "Thanks for your interest!")

# Schedule reply for tomorrow 9 AM
from datetime import datetime, timedelta
tomorrow_9am = (datetime.now() + timedelta(days=1)).replace(hour=9, minute=0).isoformat() + 'Z'
send_reply(12345, "abc-124", "Following up on our previous conversation", schedule_time=tomorrow_9am)
```

```javascript JavaScript
const API_KEY = 'YOUR_API_KEY';

async function sendReply(campaignId, emailStatsId, body, options = {}) {
  const payload = {
    campaignId,
    emailStatsId,
    emailBody: body,
    addSignature: options.addSignature !== false,
    ...options
  };
  
  const response = await fetch(
    `https://server.smartlead.ai/api/v1/email-campaigns/send-email-thread?api_key=${API_KEY}`,
    {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify(payload)
    }
  );
  
  return response.json();
}

// Send reply
await sendReply(12345, 'abc-123', 'Happy to help!');
```
