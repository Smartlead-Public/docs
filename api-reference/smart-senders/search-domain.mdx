<RequestExample>


```python Python
import requests

API_KEY = "YOUR_API_KEY"

response = requests.get(
    "https://server.smartlead.ai/api/smart-senders/search-domain",
    params={"api_key": API_KEY}
)

result = response.json()
print(result)
```

```javascript JavaScript
const API_KEY = 'YOUR_API_KEY';
```
