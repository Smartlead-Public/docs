</ParamField>

## Headers

<ParamField header="Content-Type" type="string">
  `application/json` (not required for GET, but safe to include)
</ParamField>



<RequestExample>

```bash cURL
curl "https://smart-senders.smartlead.ai/api/v1/smart-senders/get-domain-list?api_key=YOUR_API_KEY"
```

```python Python
import requests

API_KEY = "YOUR_API_KEY"

response = requests.get(
    "https://smart-senders.smartlead.ai/api/v1/smart-senders/get-domain-list",
    params={"api_key": API_KEY}
)

result = response.json()
print(result)
```

```javascript JavaScript
const API_KEY = 'YOUR_API_KEY';

const response = await fetch(
  `https://smart-senders.smartlead.ai/api/v1/smart-senders/get-domain-list?api_key=${API_KEY}`
);

const result = await response.json();
console.log(result);
```
