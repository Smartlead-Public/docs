---
title: "Push Prospects to List"
api: "POST https://server.smartlead.ai/api/v1/search-email-leads/push-to-list"
description: "Push enriched Smart Prospect leads to a lead list"
---

<Note>
After enriching leads through Smart Prospect, push them to a lead list for campaign use. Provide either a filter ID (from a saved search) or specific lead IDs.
</Note>

## Query Parameters

<ParamField query="api_key" type="string" required>
  Your SmartLead API key
</ParamField>

## Request Body

<ParamField body="listId" type="number" required>
  Target lead list ID to push prospects into
</ParamField>

<ParamField body="filterId" type="number">
  ID of a saved search filter. Provide either `filterId` or `leadIds`, not both.
</ParamField>

<ParamField body="leadIds" type="array">
  Array of prospect lead ID strings. 1-10,000 items. Provide either `filterId` or `leadIds`, not both.
</ParamField>

<ParamField body="verification_status" type="string">
  Filter by email verification status before pushing
</ParamField>

<ParamField body="catch_all_status" type="string">
  Filter by catch-all status before pushing
</ParamField>

<RequestExample>

```bash cURL
curl -X POST "https://server.smartlead.ai/api/v1/search-email-leads/push-to-list?api_key=YOUR_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "listId": 500,
    "leadIds": ["prospect_abc123", "prospect_def456"]
  }'
```

</RequestExample>
