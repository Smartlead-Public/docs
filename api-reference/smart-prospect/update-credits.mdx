---
title: "Update Prospect Credits"
api: "PATCH https://server.smartlead.ai/api/v1/search-email-leads/update-credits"
description: "Increase or decrease Smart Prospect credits for your account"
---

## Query Parameters

<ParamField query="api_key" type="string" required>
  Your SmartLead API key
</ParamField>

## Request Body

<ParamField body="amount" type="number" required>
  Number of credits to add or subtract. Must be a positive integer >= 1.
</ParamField>

<ParamField body="type" type="string" required>
  Operation type. Values: `increase` (add credits) or `decrease` (subtract credits)
</ParamField>

<RequestExample>

```bash cURL
curl -X PATCH "https://server.smartlead.ai/api/v1/search-email-leads/update-credits?api_key=YOUR_KEY" \
  -H "Content-Type: application/json" \
  -d '{"amount": 1000, "type": "increase"}'
```

</RequestExample>
