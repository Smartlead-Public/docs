---
title: "Bulk Enrich Leads"
api: "POST https://server.smartlead.ai/api/v1/lead-enrichment/fetch-leads"
description: "Submit a bulk lead enrichment request with results delivered to an optional webhook or stored for retrieval"
---

<Note>
Bulk enrichment for large datasets. Supports file-based input via AWS URLs. Results can be optionally delivered via webhook.
</Note>

## Request Body

<ParamField body="input" type="object" required>
  Input data or filter criteria for bulk enrichment
</ParamField>

<ParamField body="name" type="string">
  Name for this bulk enrichment batch
</ParamField>

<ParamField body="type" type="object" required>
  Enrichment type configuration
</ParamField>

<ParamField body="leadEnrichmentId" type="number">
  Existing enrichment batch ID to continue
</ParamField>

<ParamField body="awsFileUrl" type="string">
  AWS S3 URL to a CSV file containing leads to enrich
</ParamField>

<ParamField body="webhookUrl" type="string">
  Webhook URL for result delivery
</ParamField>

<RequestExample>

```bash cURL
curl -X POST "https://server.smartlead.ai/api/v1/lead-enrichment/fetch-leads?api_key=YOUR_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "input": {"filters": {"industry": "Technology", "company_size": "50-200"}},
    "name": "Tech Companies Bulk Enrichment",
    "type": {"people": true, "company": true},
    "webhookUrl": "https://yourapp.com/webhook/bulk-enrichment"
  }'
```

</RequestExample>
