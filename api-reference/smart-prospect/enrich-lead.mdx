---
title: "Enrich Leads (API Key)"
api: "POST https://server.smartlead.ai/api/v1/lead-enrichment/fetch"
description: "Enrich lead data using an API key with webhook callback for results"
---

<Note>
Submit a lead enrichment request via API key authentication. Results are delivered asynchronously to your specified webhook URL.
</Note>

## Query Parameters

<ParamField query="apiKey" type="string" required>
  Your SmartLead API key (note: parameter name is `apiKey`, not `api_key`)
</ParamField>

## Request Body

<ParamField body="input" type="object" required>
  Input data for enrichment (e.g., company domain, LinkedIn URL, email)
</ParamField>

<ParamField body="name" type="string">
  A name to identify this enrichment request
</ParamField>

<ParamField body="type" type="object" required>
  Enrichment type configuration specifying what data to look up
</ParamField>

<ParamField body="leadEnrichmentId" type="number">
  Existing enrichment ID to update/continue
</ParamField>

<ParamField body="webhookUrl" type="string" required>
  URL to receive enrichment results when processing completes
</ParamField>

<RequestExample>

```bash cURL
curl -X POST "https://server.smartlead.ai/api/v1/lead-enrichment/fetch?apiKey=YOUR_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "input": {"domain": "acme.com"},
    "name": "ACME Corp Enrichment",
    "type": {"people": true, "company": true},
    "webhookUrl": "https://yourapp.com/webhook/enrichment"
  }'
```

</RequestExample>

## Response Codes

<ResponseField name="200" type="Success">
  Request successful
</ResponseField>

<ResponseField name="400" type="Bad Request">
  Invalid request parameters or malformed request body
</ResponseField>

<ResponseField name="401" type="Unauthorized">
  Invalid or missing API key
</ResponseField>

<ResponseField name="422" type="Validation Error">
  Request validation failed. Check parameter types and constraints.
</ResponseField>

<ResponseExample>

```json 200 - Success
{
  "ok": true,
  "data": {
    "enrichment_id": 12345,
    "status": "processing",
    "message": "Enrichment started. Results will be sent to your webhook URL."
  }
}
```

```json 401 - Unauthorized
{
  "message": "Invalid API Key"
}
```

</ResponseExample>
