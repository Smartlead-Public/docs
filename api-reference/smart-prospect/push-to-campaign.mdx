---
title: "Push Prospects to Campaign"
api: "POST https://server.smartlead.ai/api/v1/search-email-leads/push-to-campaign"
description: "Push enriched Smart Prospect leads directly to a campaign"
---

<Note>
Push prospects directly to a campaign for immediate outreach. Supports CSV settings for block list and duplicate handling.
</Note>

## Query Parameters

<ParamField query="api_key" type="string" required>
  Your SmartLead API key
</ParamField>

## Request Body

<ParamField body="campaignId" type="number" required>
  Target campaign ID
</ParamField>

<ParamField body="filterId" type="number">
  ID of a saved search filter. Provide either `filterId` or `leadIds`.
</ParamField>

<ParamField body="leadIds" type="array">
  Array of prospect lead ID strings. 1-10,000 items.
</ParamField>

<ParamField body="verification_status" type="string">
  Filter by email verification status
</ParamField>

<ParamField body="catch_all_status" type="string">
  Filter by catch-all status
</ParamField>

<ParamField body="csvSettings" type="object">
  Campaign import settings
</ParamField>

<ParamField body="csvSettings.ignoreGlobalBlockList" type="boolean">
  Skip global block list check
</ParamField>

<ParamField body="csvSettings.ignoreUnsubscribeList" type="boolean">
  Skip unsubscribe list check
</ParamField>

<ParamField body="csvSettings.ignoreDuplicateLeadsInOtherCampaign" type="boolean">
  Allow leads already in other campaigns
</ParamField>

<ParamField body="csvSettings.ignoreCommunityBounceList" type="boolean">
  Skip community bounce list check
</ParamField>

<RequestExample>

```bash cURL
curl -X POST "https://server.smartlead.ai/api/v1/search-email-leads/push-to-campaign?api_key=YOUR_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "campaignId": 12345,
    "filterId": 67,
    "csvSettings": {
      "ignoreGlobalBlockList": false,
      "ignoreDuplicateLeadsInOtherCampaign": true
    }
  }'
```

</RequestExample>
