```python Python
import requests

API_KEY = "YOUR_API_KEY"
campaign_id = 123
url = f"https://server.smartlead.ai/api/v1/campaigns/{campaign_id}/leads"

leads = [
    {
        "first_name": "John",
        "last_name": "Doe",
        "email": "john.doe@example.com",
        "company_name": "Acme Corp",
        "custom_fields": {
            "job_title": "CEO",
            "industry": "Technology"
        }
    },
    {
        "first_name": "Jane",
        "last_name": "Smith",
        "email": "jane.smith@techco.com",
        "company_name": "Tech Co",
        "custom_fields": {
            "job_title": "CTO"
        }
    }
]

response = requests.post(
    url,
    params={"api_key": API_KEY},
    json={"lead_list": leads}
)

result = response.json()
print(f"Added: {result['added_count']}, Skipped: {result['skipped_count']}")
```

## Custom Fields & Personalization

Custom fields are key-value pairs that can be used for email personalization. You can add up to 200 custom fields per lead.

### Common Custom Fields

- `job_title`: "CEO", "CTO", "VP of Sales"
- `industry`: "Technology", "Healthcare", "Finance"
- `company_size`: "1-10", "11-50", "51-200"
- `annual_revenue`: "$1M-$5M", "$5M-$10M"
- `location`: "New York, USA"
- `phone`: "+1234567890"

### Using Custom Fields in Emails

```
Subject: Quick question for the {{job_title}} at {{company_name}}

Hi {{first_name}},

I noticed {{company_name}} is in the {{industry}} industry...
```

## Lead Validation

Leads are validated for:

1. **Email Format**: Must be valid email format
2. **Duplicates**: Checked against existing leads in the campaign
3. **Block List**: Checked against global and client-specific block lists
4. **Unsubscribe List**: Previously unsubscribed emails are skipped (unless ignored)
5. **Community Bounce List**: Known bounced emails are skipped (unless ignored)

## Limits and Best Practices

<AccordionGroup>
  <Accordion title="Maximum Leads Per Request">
    You can add up to 400 leads per API request. For larger imports, split into batches.
  </Accordion>

  <Accordion title="Custom Fields Limit">
    Each lead can have up to 200 custom fields. Keep field names consistent across all leads.
  </Accordion>

  <Accordion title="Duplicate Handling">
    By default, duplicate emails within the same campaign are skipped. Use `ignore_duplicate_leads_in_other_campaign` to allow the same lead in multiple campaigns.
  </Accordion>

  <Accordion title="Batch Import Tips">
    - Import in batches of 200-400 for optimal performance
    - Add delays between batches to avoid rate limiting
    - Verify emails before importing for better deliverability
  </Accordion>
</AccordionGroup>

## Common Issues

### Issue: All leads are being skipped

**Solution**: Check if:
- Leads already exist in the campaign
- Emails are in the global block list
- Email format is valid
- Campaign status allows adding leads

### Issue: Custom fields not appearing in emails

**Solution**: 
- Ensure field names match exactly between lead data and email template
- Use lowercase with underscores (e.g., `job_title` not `Job Title`)
- Check that fields are properly enclosed: `{{job_title}}`
