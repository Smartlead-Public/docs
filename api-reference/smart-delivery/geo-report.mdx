## Query Parameters

  Your SmartLead API key
</ParamField>



<RequestExample>

```bash cURL
curl -X POST "https://server.smartlead.ai/api/smart-delivery/geo-report?api_key=YOUR_KEY" \
  -H "Content-Type: application/json" \
  -d '{"key": "value"}'
```

```python Python
import requests

API_KEY = "YOUR_API_KEY"

response = requests.post(
    "https://server.smartlead.ai/api/smart-delivery/geo-report",
    params={"api_key": API_KEY},
    json={"key": "value"}
)

result = response.json()
print(result)
```

```javascript JavaScript
const API_KEY = 'YOUR_API_KEY';

const response = await fetch(
  `https://server.smartlead.ai/api/smart-delivery/geo-report?api_key=${API_KEY}`,
  {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({ key: 'value' })
  }
);

const result = await response.json();
console.log(result);
```
